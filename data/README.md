# Data Assets Directory

This directory stores inputs and evaluation logs. To avoid bloating the repository, files in `inputs/` and `outputs/` are ignored by Git (except for their `.gitkeep` files).

## Folders
- `inputs/`: Place raw benchmark datasets (e.g. `questions.jsonl`, `test_data.csv`) here.
- `outputs/`: Saves generated LLM logs, inference outputs, and metric results.
